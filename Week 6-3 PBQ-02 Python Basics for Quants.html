Functions: def / return

def f(x, y, z):
    k = x**2 + y**2 - z
    return k        # NOT print!
f(2,3,1)  # -> 12
neg = lambda x: x<0   # anonymous
⚠ print ≠ return. print shows a value; return sends it out so you can use it downstream.
5

Why NumPy? lists ≠ maths

exprresult
[1,2]+[6,7][1,2,6,7] concatenates
np.array([1,2])+np.array([6,7])[7,9] element-wise
Speed (%timeit): max() on a 1000-list ≈ 7.5 µs vs array ≈ 1.5 µs — ~5× faster (NumPy runs in C).
