① AI ⊃ ML ⊃ DL

  • AI — machines taking human-like decisions, built from collected natural intelligence (labelled history). Doctor-in-a-town analogy.
  • ML — algorithms that learn the association input → output from data.
  • DL — a subset of ML inspired by the brain (neural nets, lecture 3).
AI helps us decide; it does not replace natural intelligence — it amplifies it.

④ Logistic regression: line → sigmoid → probability

A straight line fits a numeric target, but fails on a binary 0/1 target. Push the linear score through the sigmoid (logit) to bend it into an S-curve bounded in (0,1):

z = α·x + β  →  σ(z) = 1 / (1 + e−z)
z−4−20+2+4
σ(z)0.0180.1190.5000.8810.982
Output is a probability; threshold 0.5 (default, tunable) → class 1 if P>0.5. It is a classifier, not a regressor. >2 classes → multinomial/softmax.

⑤ Decision trees → Gini impurity → random forests

A · A tree asks yes/no questions

Each node splits on one feature + threshold to make the branches purer. The lecture's 30-person "Fit?" set splits on Age<27. Purity is scored by the Gini impurity:

Gini = 1 − Σ pj2
Alternatives: entropy/information gain (ID3, C4.5) and classification error 1−max(pj). All measure impurity; the tree greedily picks the split that lowers it most.

C · Overfitting → random forest
  • Occam's razor — between two equally-accurate models, prefer the simpler (shallower) tree.
  • Pruning — pre (stop growing early) or post (trim bottom-up) to fight overfitting.
  • Random forest = ensemble of trees, each on a bootstrap sample + a random feature subset per node; final class by majority vote → generalises better.

⑨ Gotchas that quietly inflate accuracy

  • 🚫 Shuffling a time series in the split — trains on the future. Use shuffle=False.
  • 🚫 Fitting the scaler on test data — leaks future μ,σ. transform test only.
  • 🚫 Predicting on unscaled data when the model trained on scaled data.
  • ✅ shift(-1) to build the target is fine — it's dataset construction, done before training.
  • ✅ Set random_state everywhere for reproducible results.
Gini = 1 − Σ pj2
σ(z) = 1 / (1 + e−z)  ·  P>0.5 → 1
z-score = (x − μ) / σ
