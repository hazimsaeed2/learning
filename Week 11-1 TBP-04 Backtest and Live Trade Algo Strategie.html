⭐ CORE EXAMPLE — a linear-regression "swing" model

Predict tomorrow's % move from today's % move via linear regression. On NIFTY (SPY stand-in) the coefficient is −0.08 → a "swing world" (up today → down tomorrow) — but the correlation is only ~0.08, statistically very weak. The swing backtest loses −3.4% vs buy-and-hold +58.3%. Structure is real; the edge is thin.

⑦ Trend vs Swing & the model

Trend (momentum)Swing (mean-revert)
up → keep up. Buy&hold = ultimate trendup too much → falls back
# feature engineering by shifting
close_yest = Close.shift(1)
chg_y2t = (Close-close_yest)/close_yest  # predictor
chg_t2m = chg_y2t.shift(-1)              # TARGET
LinearRegression().fit(X=chg_y2t, y=chg_t2m)

Negative coef → swing world. Swing rule: position = −sign(today's move).
