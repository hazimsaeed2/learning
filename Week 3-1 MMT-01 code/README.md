# Week 3-1 MMT-01 Execution Strategy I code

Open `Week 3-1 MMT-01 Execution Strategy I_practice.ipynb` first. It is the
original practice notebook and has been left in place.

Open `Week 3-1 MMT-01 Execution Strategy I_resource_addendum_practice.ipynb`
for the added workbook, FAQ, and reading-based validation exercises.

The addendum notebook uses local CSV files transcribed from the official in-class
Excel workbook:

- `mmt01_orderbook_abc.csv`
- `mmt01_depth_structure.csv`
- `mmt01_vwap_trades.csv`
- `mmt01_pro_rata_sell_10.csv`
