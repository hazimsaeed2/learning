🎲 WHAT — random walk price ranges

Returns ~ normal & independent → ranges from μ ± kσ. 68% in ±1σ, 95% in ±2σ.

NIFTY today18,065
daily mean / std0.046% / 1.089%
next-day 68% (±1σ)17,877 – 18,271
next-day 95% (±2σ)17,684 – 18,471

Model returns, not prices: prices are chained to yesterday; returns are free & scale-independent.

📏 HOW — annualize volatility (√T)

Returns add; std doesn't. Variance adds (indep returns) → σ_T = σ₁√T.

Toy: 1%/day → 1%·√252 ≈ 16%/yr (NOT 252%).

⚠ WHY — simple vs log returns: the drift trap

Averaging simple returns overstates compounding. Fund years 15, 23, 26, −11, −5%:

STEP TABLE ④ — start with 100

method5-yr
simple avg 9.60% → 100·(1.096)⁵158.14
actual 100·Π(1+r)150.69
Fix A: log avg → 100·e^(5·avg)150.69 ✓
Fix B: drift μ−σ²/2≈148–151

THE RULE

Use log returns (problem vanishes) — OR subtract drift = μ − σ²/2 from simple returns (from Itô's lemma). Some regulators ban advertising the naive simple average.

Why log adds: cumulative log return = sum of daily logs; simple cumulative = product of (1+r). Logs are easier.

🧪 HOW — Monte Carlo → range, VaR & expected shortfall

Draw normal returns =NORMINV(RAND(),μ,σ), roll Pₜ=Pₜ₋₁·e^r. 5000 paths × 20 days.

STEP TABLE ⑤ — NIFTY 20-day risk (5%)

MC 95% range16,544 – 20,012
closed-form 95%16,538 – 20,096
5% VaR floor (P5)16,796
expected shortfall (mean worst 5%)16,456

3 methods agree. VaR = "95% sure above 16,796". ES = "if it breaks, expect 16,456" (how bad is bad).

📈 APPLY — Bollinger Bands (mean-reversion)

20-day SMA ± 2σ. upper = sma + 2·std, lower = sma − 2·std.

Strategy: price closes below lower → BUY; closes above upper → SELL/exit. Bet on reversion to mean.

⚠ FAT TAILS: theory says ~5% of closes fall outside ±2σ; real NIFTY ~9%. Crashes (1929, 2020) blow past 3σ → model out to 4–5σ. Normal is only an approximation.

CONTINUATION ADDENDUM · transcript-only operating notes

This continuation preserves the existing sheet and adds practical lecture comments that were not yet explicit in the validation addendum.

NO LOOK-AHEAD TRAP

The repeated average column is an Excel convenience, not a signal known on day 1. The decision is made after the sample window closes, then rebalanced on the chosen schedule.

LONG-ONLY FRONTIER

The 0% to 100% weight grid is a long-only investing exercise. Extending weights below 0% means shorting one stock and using proceeds to buy the other, which is a different trading problem.

EXCEL DECODERS

$A$1 locks a reference. 6.86E-05 means 6.86 x 10^-5 = 0.0000686. Use sample formulas when the data is a return sample.

LECTURE METHOD VS PRACTICAL CONTROL

TopicLecture spreadsheet shortcutCorrect practical interpretation
Mean beside every rowSame full-window average copied downUse only after the full historical window is available; in backtests, row t must use data up to t only.
Portfolio weights101 portfolios: 0%, 1%, ..., 100%Long-only grid. Negative weights are short positions and need different assumptions, costs, and constraints.
Covariance formulaSUM((x-xbar)(y-ybar))/(n-1)Sample covariance. In modern Excel prefer COVARIANCE.S; population covariance divides by n.
Expected shortfallAVERAGEIF(paths,"<"&VaR)Average only the outcomes below the VaR floor; use a cell reference so the threshold updates when simulations refresh.

MODEL FAILURE CHECKLIST

QuestionWhy it matters
Are returns independent?If not, the square-root-of-time and random-walk assumptions weaken. Check autocorrelation or use time-series models such as ARIMA.
Did volatility regime change?A shock can sit outside the old 95% range before rolling volatility catches up.
Are tails skewed or fat?NORMINV ignores skew and kurtosis; Monte Carlo can use a different return distribution if the normal model fails.
Is Excel still the right tool?Excel is fine for quick checks; Python is safer for large data, repeated research, and high-frequency workflows.

Correction-by-addition: if any earlier wording sounds like a universal rule, read it with these constraints. The lecture's Bollinger, VaR, and Monte Carlo examples are normal-model demonstrations, not guarantees that price stays inside the band or VaR floor.
