CORE EXAMPLE — print it like a pro

A bare number tells you nothing; labelled output tells a story. Three ways to print Intel's market cap — same result.

# style 1 — comma-separated
print("Mkt cap of", firm_name, "is US$", market_cap)
# style 2 — f-string (cleanest)
print(f"ticker {ticker}  price US$ {last_price}")
# style 3 — .format with {index:type.precision}
print("{0:s} has {1:.1f} million shares".format(firm_name, shares))
Output → Intel Corporation has 4159.8 million shares  (105.95 bn ÷ 25.47 ÷ 10⁶)

NumPy — why arrays beat lists

Arrays are homogeneous and do element-wise maths. This single difference is why every back-test uses arrays.

coderesultmeaning
[1.65,1.8] * 2[1.65,1.8,1.65,1.8]list → duplicated!
np.array([1.65,1.8]) * 2[3.3, 3.6]array → scaled ✓
 
import numpy as np
x = np.random.normal(loc=0, scale=1, size=10000)  # 10k pts, 1 line
idx = np.where(vol < 150)        # indices meeting a condition
vol[idx]                       # the values there
np.random.normal: loc=mean, scale=std, size=count. Math funcs (np.mean, np.median) live in NumPy, not on lists.

Addendum: official workflow mechanics

Jupyter and environment workflow
  • Esc command mode, Enter edit mode, Shift+Enter run current cell.
  • help(obj), obj?, tab completion, %who, and %whos inspect code and memory.
  • conda is strong for environments and scientific packages; pip installs Python packages.
  • AI can explain errors and examples, but trading logic and signal alignment still need human verification.
Python and pandas gotchas
  • set means unordered unique values; use it for de-duplication and membership checks.
  • append(x) adds one object; extend(xs) adds each item from xs.
  • .loc is label-based; .iloc is integer-position based.
  • None and NaN are missing values, not zero. Rolling signals create warm-up NaN rows.
