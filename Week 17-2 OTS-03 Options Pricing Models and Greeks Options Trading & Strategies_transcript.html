PREMIUM DECOMPOSITION

Premium = Intrinsic Value + Extrinsic Value

Intrinsic = max(S − X, 0) for calls ← exercise value NOW
= max($100 − $90, 0) = $10

Extrinsic = $4 ← what you pay for TIME to act
• Decays daily (Theta Greek measures this)
• Goes to $0 at expiry if ATM or OTM
• You're paying $4 for the right to act for 30 days

PAYOFF vs P&L

Call Payoff = max(S − X, 0) ← ignores cost
Call P&L = max(S − X, 0) − Premium ← real profit

Example: S=$110, X=$90, Premium=$14
Payoff = max(110−90, 0) = $20
P&L = $20 − $14 = +$6 profit

Example: S=$95, X=$90, Premium=$14
Payoff = max(95−90, 0) = $5
P&L = $5 − $14 = −$9 loss

2️⃣ OPTION COMBINATIONS — Hedging, Spreads, Straddles & Why Overdoing Kills Profits

PROTECTIVE PUT (Stock + Long Put)

Position: Long stock at $90 + Buy $90 put for $10

If stock falls to $70:
Stock loss = $90 − $70 = −$20
Put payoff = $90 − $70 = +$20
Net = $0 − $10 premium = −$10

If stock rises to $120:
Stock gain = $120 − $90 = +$30
Put payoff = $0 (worthless)
Net = $30 − $10 premium = +$20

Max loss = $10 (premium only) ✅
Breakeven = $90 + $10 = $100
Profit starts above $100

💡 KEY: Hedging LIMITS losses but NEVER eliminates them. You always pay a cost (the premium). Think of it like insurance — you pay to cap the downside.

BULL CALL SPREAD

Position: Buy $90 call (pay $10) + Sell $120 call (receive $5)
Net cost = $10 − $5 = $5 (50% cheaper!)

If stock at $110 on expiry:
Long 90 call: $110−$90 = +$20
Short 120 call: $0 (worthless)
Net = $20 − $5 cost = +$15 ✅

If stock at $200 on expiry:
Long 90 call: $200−$90 = +$110
Short 120 call: −($200−$120) = −$80
Net = $110 − $80 − $5 = +$25 (capped!)

Max profit = ($120−$90) − $5 = $25
Max loss = $5 (net premium paid)

When to use: You're bullish but believe stock won't go above $120 → save premium by selling the upside you don't expect.

STRADDLE vs STRANGLE

FeatureStraddleStrangle
PositionATM call + ATM putOTM call + OTM put
CostExpensiveCheaper
Profit zoneAny direction beyond premiumOnly on extreme moves
Use caseUnknown direction, expect big moveSame, but budget-friendly

When: Before Fed rate decisions, budget announcements, earnings — you don't know direction but expect high volatility. Can't do this with equities (hands tied if you don't know direction).

⚠️ 1×2 CALL SPREAD TRAP: Buying 1 call + selling 2 higher calls to "finance" the position. If stock goes way up (your direction was RIGHT), you LOSE money because the extra short call creates unlimited downside. Never overuse combinations just to collect premium.

3️⃣ PUT-CALL PARITY — The Fundamental Equation That Ties Everything Together

THE EQUATION

C − P = S − X × e−rT

CCall premium
PPut premium
SSpot price of underlying
XStrike price
rRisk-free rate (from 91-day Treasury)
TTime to expiry (in years)

PROOF — Portfolio Value at Expiry

Portfolio = Long Put + Short Call + Long Stock
Initial value = P(x) − C(x) + S

If S < X at expiry (market fell):
Long put payoff = X − S
Short call payoff = 0 (worthless)
Stock value = S
Total = (X − S) + 0 + S = X

If S > X at expiry (market rose):
Long put payoff = 0 (worthless)
Short call payoff = −(S − X)
Stock value = S
Total = 0 − (S − X) + S = X

Both = X → portfolio value is CONSTANT
regardless of direction → parity holds ✅

KEY IMPLICATIONS

ImplicationWhy It Matters
Change in call → change in putIf volatility spikes, BOTH premiums change (not just calls)
Know C → calculate PBSM gives C, put-call parity gives P
Find strike where C = PSet C−P=0 → X = S × erT
Combinations workSpreads, straddles, hedging all possible BECAUSE parity holds
If violated → arbitrageMarket makers quickly close any gap

WORKED EXAMPLE

S = $100, X = $90, r = 0%, T = 30 days
C − P = S − X × e−rT
C − P = 100 − 90 × e0 = 100 − 90 = $10

If C = $14 → P = $14 − $10 = $4
If C = $10 → P = $10 − $10 = $0

⚠️ AMERICAN OPTIONS: Put-call parity does NOT hold for American options. T is not constant (can exercise anytime). Therefore BSM (Black-Scholes-Merton) doesn't work for American options either. American options need different mathematical methods (e.g., binomial trees).

MODEL-INDEPENDENT RELATIONSHIPS

American options > European optionsMore optionality = more cost
Longer expiry > shorter expiryMore time = more cost
Call premium ≤ underlying priceIf C > S → free arbitrage
Put premium ≤ strike priceSame logic
Lower strike call > higher strike callDeeper ITM = more premium

4️⃣ BLACK-SCHOLES MODEL — The Closed-Form Equation for Options Pricing

THE EQUATION

C = S × N(d₁) − X × e−rT × N(d₂)

d₁ = [ln(S/X) + (r + σ²/2) × T] / (σ × √T)
d₂ = d₁ − σ × √T

N()Cumulative standard normal distribution function
N(d₁)Hedge ratio (delta) — how many shares to hedge
N(d₂)Probability of finishing ITM (theoretical, risk-neutral only)
σAnnualized volatility (std dev of daily returns × √252)
ln()Natural logarithm

Structure: At its core, BSM (Black-Scholes-Merton) is S − X (spot minus strike), but each term is multiplied by risk-adjustment factors (N(d₁), N(d₂), e−rT) that incorporate volatility, time, and interest rates.

Put from BSM: Calculate call C from BSM → plug into put-call parity → P = C − S + X × e−rT

BSM ASSUMPTIONS (Limitations)

AssumptionReality
Constant volatilityBiggest issue! Vol changes every second → premium changes too
Continuous tradingMarkets close, have illiquid hours
Log-normal distributionFat tails exist in real markets
No dividendsDividends affect pricing (modified BSM exists)
No transaction costsLot sizes, commissions, taxes are real
Risk-free rate constantRates change (recalibrate inputs)
European options onlyFixed T required (American: variable T)
Efficient marketsNot always true

PYTHON — CALCULATE PREMIUM WITH mibian

# pip install mibian              # Install if needed
import mibian                     # Options pricing library

# Define inputs
stock_price = 100                 # Current spot (S)
strike_price = 90                 # Strike (X)
interest_rate = 0                 # Risk-free rate (r) in %
days_to_expiry = 30               # Days until expiry (T)
volatility = 10                   # Annualized vol (σ) in %

# Calculate call premium using BSM
bs = mibian.BS(                   # Black-Scholes model
    [stock_price, strike_price,   # S, X
     interest_rate, days_to_expiry], # r, T
    volatility=volatility)        # σ

call_price = bs.callPrice         # Theoretical call value
put_price = bs.putPrice           # Theoretical put value
# call_price ≈ $10.03
# put_price  ≈ $0.03

💡 WHY NOT RETURNS? Returns are about the PAST. Options are about the FUTURE. In the lecture's pricing discussion, the direct focus was on volatility rather than historical return series.

5️⃣ OPTIONS GREEKS — Partial Derivatives of Premium: How Each Factor Moves Your Position

DELTA (Δ) — Impact of Price Change

Δ = ∂Premium / ∂Spot = N(d₁) for call
Δ_put = N(d₁) − 1
Δ_call − Δ_put = 1 (from put-call parity)

Δ_callΔ_putMeaning
0.9−0.1Deep ITM call: $1 up → $0.90 premium increase
0.5−0.5≈ ATM: $1 up → $0.50 premium increase
0.1−0.9Deep OTM call: $1 up → $0.10 increase

Trader jargon: "I have a 90 delta call" = delta = 0.9 = very responsive to price. Also serves as hedge ratio (how many shares to buy/sell to hedge).

GAMMA (Γ) — Stability of Delta

Γ = ∂Δ / ∂Spot (second derivative of premium)

ScenarioGammaDelta Behavior
ATM contractHIGHESTDelta changes rapidly
Deep ITM / OTMLOWDelta very stable
Near-term expiryHIGHESTExplosive delta swings
Long-term expiryLOWSmooth delta changes

Decision: Want stable delta → go deep ITM/OTM + recent expiry. Want high gamma (volatile delta) → go ATM + near-term.

THETA (Θ) — Time Decay

Θ = ∂Premium / ∂Time

ScenarioThetaDecay Speed
ATM contractHIGHESTDecays fastest
Deep ITM / OTMLOWSlower decay
Near expiryHIGHESTAccelerating decay

Jargon: "Short 500 theta" = losing $500/day from time decay. The lecture used this mainly to show how traders talk about daily time-decay exposure in dollar terms.

VEGA (ν) — Volatility Sensitivity

ν = ∂Premium / ∂Volatility

ScenarioVegaVol Sensitivity
ATM contractHIGHESTMost affected by vol change
Deep ITM / OTMLOWLess affected
Longer expiryHIGHERMore time for vol to impact
Near expiryLOWLess time for vol to matter

Jargon: "Long 1000 Vega" = if volatility goes up 1%, I make $1,000. CRUCIAL for volatility trading (OTS-04 focus).

RHO (ρ) — Interest Rate Impact

Minor Greek. ρ = ∂Premium/∂Rate. Relevant only for long-dated positions (months/years) since interest rates change infrequently. Use 91-day Treasury rate.

PORTFOLIO-LEVEL GREEKS (Linear!)

All Greeks are additive across positions. Sum individual Greeks to get portfolio Greek:

Portfolio Δ = Δ₁ + Δ₂ + ... + Δₙ
Portfolio Γ = Γ₁ + Γ₂ + ... + Γₙ
Same for Θ, ν, ρ

SECOND-ORDER GREEKS

Vanna∂Δ/∂Volatility — how delta changes with vol
∂Vega/∂TimeHow Vega changes as you near expiry

You can create custom Greeks by taking partial derivatives of BSM with respect to any variable. Quants do this for specific trading strategies.

💡 PRACTICAL BSM USE: You don't need BSM to find premium (just look at the option chain). You need BSM to calculate Greeks — and Greeks tell you HOW to manage risk, WHEN to hedge, and WHICH contract to pick. That's the real value.

🎯 KEY TAKEAWAYS

1. Premium = Intrinsic + Extrinsic. Intrinsic is the exercise-now value (S−X for calls). Extrinsic is the time value — what you pay for the right to act. Theta measures how fast the extrinsic portion decays daily, and ATM contracts decay fastest.

2. Put-call parity (C−P = S−Xe−rT) is the foundation. It ties calls and puts together so that combinations (spreads, straddles, hedging) work predictably. It was presented here for European options; American-style contracts need different mathematical treatment.

3. BSM's real value isn't computing premium — it's computing Greeks. You can see premium in any option chain. But Greeks (Delta, Gamma, Theta, Vega) tell you HOW your premium changes as price/time/volatility shift. All Greeks are partial derivatives of the BSM equation.

4. ATM is the epicenter of all Greeks. Highest gamma (delta most volatile), highest theta (decays fastest), highest vega (most sensitive to vol). Choose ITM/OTM + long/short expiry based on which Greek behavior your strategy needs. Don't buy the cheapest option — buy the right strike.

THE MASTER FORMULA: C = S×N(d₁) − X×e−rT×N(d₂) — every options Greek (Δ, Γ, Θ, ν, ρ) is a partial derivative of this equation. Understand what each Greek measures, then pick contracts where the Greek behavior matches your trading objective.

THE ONE THING TO NEVER FORGET: Options are complex but trading them is simple — IF you think in terms of Greeks. Delta tells you price exposure, Gamma tells you delta stability, Theta tells you time cost, and Vega tells you volatility bet. Before any options trade, know all four numbers for your position.
