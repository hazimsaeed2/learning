③ Simple vs log returns

Close but not equal: on our NSE data the max gap was 0.00923. Log returns always slightly understate the simple return; the gap grows as |return| grows. (Like kg vs lb — same weight, different unit.)

④ Match the tool to the return

# simple returns -> cumprod
equity = (1 + r).cumprod() - 1

# log returns -> cumsum
cum_log = log_r.cumsum()

You cannot mix them: simple needs the product rule, log needs the sum rule. Pick one unit and stay in it.

Why log? additive (humans prefer adding), and nice statistical properties for models.

⑤ .shift() and look-ahead bias

shift(1) moves a column down (yesterday's value lands on today's row); shift(-1) pulls the future up. We shift the position by 1 so a signal computed on a bar is acted on the next bar.

⚠ Look-ahead bias: using information earlier than you could have had it. Forget the shift and your back-test "buys" using a close it hasn't seen yet — fake profits.

⑥ Vectorized vs event-driven back-testing

Vectorized (this lecture)Event-driven
Processeswhole matrix at once (NumPy)one candle/event at a time (a loop)
Speedvery fastslow, computationally costly
Realismoptimistic — ignores fills/slippagemodels unfilled/partial orders, slippage, order book
Use forscreening many ideas (15 → 2)final validation of survivors
An "event" = each new candle. With 15-min bars over a 6-hr day, the engine re-checks rules 24 times/day. Tools: Blueshift (free, backtest + live).

⑦ The vectorized SMA strategy — step by step

df["cc"]  = df.Close.pct_change()          # returns
df["SMA"] = df.Close.rolling(12).mean()    # indicator
df["pos"] = np.where(df.Close>df.SMA,1,0)  # rule
df["pos"] = df["pos"].shift(1)             # no look-ahead
df["strat"] = df.pos * df.cc               # strategy ret
eq = (1 + df.strat).cumprod()              # equity curve
total = eq.iloc[-1] - 1                     # final return

Honest verdict: the strategy is a dead heat with buy-and-hold (54.46% vs 54.43%) despite being invested ~64% of the time — no real edge. That's why we back-test: most simple ideas don't beat the benchmark.

⑧ Position-vector logic

A run of 1s = stay invested; the moment it flips to 0 you sell. Because pct_change assumes you bought on the previous candle, you shift the position down by one to line the trade up with the return it earns. Multiplying position × cc_returns zeroes out the bars you're flat.
