DERIVING ROLL RETURN (Arbitrage Argument)

Two portfolios, $1M cash, MUST have same total return:

PortfolioHoldingsTotal Return
A: Buy SPY ETF$1M → SPY (cash gone)SPX return + Dividend
B: Buy ES Futures$1M as margin (cash earns interest)SPX return + Roll + Interest

SOLVING FOR ROLL RETURN:

SPX + Dividend = SPX + Roll + Interest

Dividend = Roll + Interest

Roll = Dividend − Interest

WHY futures have no funding cost: When you buy ES futures, your $1M stays in the account as margin collateral — it earns interest. Unlike buying SPY where your cash disappears.

PESO FUTURES (Homework Answer)

Same arbitrage logic, two currencies:

Roll(MXN futures) = MXN interest rate − USD interest rate

(Interest rate differential between two countries)

Total Futures Return = Spot Return + Roll Return

💡 TIME IS YOUR FRIEND: Time is the friend of roll return (accumulates) and the enemy of spot return (cancels out). This is why hold-and-wait works for futures momentum.

4️⃣ FORWARD CURVE (TERM STRUCTURE) — Estimating γ from Data

The Futures Price Model

ALGEBRAIC MODEL:

F(t, T) = exp(α·t − γ·T + const)

Take log: log F(t,T) = α·t − γ·T + const

This is a linear model in two variables (t, T)

Two views of the same data:

Graph TypeX-axisEach line isSlope
Time seriesTime (t)One contractα + γ (total)
Forward curveTime-to-expiry (T)Snapshot in time−γ (neg. roll)

⚠️ CARDINAL SIN OF DATA SCIENCE: Never fit a linear regression without plotting a scatter plot first. If X,Y isn't linear and you force it, you get a meaningless number.

WHY Log Returns (Not Net Returns)

PropertyLog ReturnNet Return
Additive?YES: 1% + 2% = 3%NO: compounding
Range(−∞, +∞)[−1, +∞)
Normal dist?Possible (in theory)Impossible

WHY net return can't be normal: Price can't go below 0, so net return ≥ −1. Normal distribution requires (−∞, +∞) range. Log(price) → −∞ as price → 0, so log return CAN span (−∞, +∞).

💡 NET RETURN FORMULA:

Net return = (P_t − P_{t-1}) / P_{t-1}

If P_t = 0: return = −P_{t-1}/P_{t-1} = −1 (minimum)

Log return = log(P_t / P_{t-1}) → −∞ when P_t → 0

5️⃣ HEDGING SPOT RETURN — Isolating Pure Roll Return via ETFs

WHAT: The Spot Return Problem

Total futures return = spot + roll. We want ONLY the roll. Spot is noise. How to remove it?

ANSWER: Buy/sell an instrument that tracks the SPOT price (not futures). That hedges out spot return, leaving pure roll return.

ETFHoldsHedge Spot?Verdict
GLDPhysical gold bars (London vault)YES ✓Tracks gold spot price
XLEEnergy stocks (Chevron, Exxon)YES ✓Companies have physical oil on balance sheet
USOOil FUTURESNO ✗Buying futures = cancels your roll return!

⚠️ USO TRAP: USO holds futures, not physical oil. If you short oil futures and long USO, you're long AND short futures exposure, so you largely cancel the roll return you were trying to isolate. XLE works because energy stocks track PHYSICAL oil via reserves on balance sheets.

WORKED EXAMPLE: Gold Arbitrage Strategy

STRATEGY: Short GC (gold futures) + Long GLD

Gold is almost always in contango (γ < 0)

Short GC → earn positive roll return

Long GLD → hedge spot return to zero

Net return ≈ pure roll return

Homework: Compute the Sharpe ratio with 2% risk-free rate subtracted (because buying GLD uses cash → has funding cost).

WHY Options Don't Work as Hedge

Options introduce additional risk factors (Greeks): theta (time decay), gamma (convexity), vega (volatility). These create additional sources of P&L noise. Just use ETFs — much simpler.

💡 CAN'T HEDGE ROLL RETURN CHANGES: You can hedge SPOT return (using ETFs). You CANNOT hedge changes in roll return sign. When the sign flips, you lose money for some days/weeks until you switch. Accept it — this is the strategy's risk.

6️⃣ FUTURES DATA — Settlement Prices, Back Adjustment & Timestamp Matching

Settlement Price ≠ Last Price

WHAT: Daily futures prices are SETTLEMENT prices, not last trade prices.

WHY: E-mini tick = $50. Last trade could be noisy. Settlement = VWAP (Volume Weighted Average Price) of last few minutes.

Edge cases: Illiquid contracts (e.g., lumber) → use bid-ask midpoint if no trades.

GC settlement time: 1:30 PM ET

US stock market close: 4:00 PM ET

Back-Adjusted Continuous Contracts

Problem: Raw futures have price GAPS at rollover (different contracts, different prices). Naïve stitching → fake P&L from gaps.

Solution: Back-adjust: shift older contracts so gaps disappear.

Two methods:

1. Additive: Add/subtract gap amount

2. Multiplicative: Multiply by ratio

(Pros/cons in Ernie's "Algorithmic Trading" book)

⚠️ CAN'T USE BACK-ADJUSTED DATA TO ESTIMATE ROLL RETURN. Back adjustment destroys the forward curve structure. Use raw contract data for γ estimation, back-adjusted for backtesting P&L.

TIMESTAMP MATCHING (Critical)

⚠️ #1 ARBITRAGE BUG: If two instruments have closing prices captured at DIFFERENT times, your arbitrage backtest is FICTITIOUS. The P&L from the time gap is not real.

WORKED EXAMPLE:

GC settlement: 1:30 PM ET

GLD close: 4:00 PM ET

→ 2.5 hour gap!

→ Must use GC intraday price AT 4:00 PM

OR use GLD intraday price AT 1:30 PM

Peso futures example: Spot at 5 PM, futures at 4 PM → 1 hour gap → "amazing" profits that are 100% fictitious.

Data vendor: CSI Data (csidata.com) — cheapest, cleanest daily futures data. The instructor also noted it historically powered Yahoo Finance data for a long time.

7️⃣ SHARPE RATIO — When To Subtract the Risk-Free Rate

THE RULE: Subtract Risk-Free Rate Only When There's Funding Cost

StrategySubtract R_f?Why
Futures only (long or short)NOCash stays in account as margin → earns interest. No funding cost.
Long/short stocks ($-neutral)NOShort side generates credit interest, long side costs debit interest → roughly cancel.
Short futures + long ETFYESBuying ETF uses cash → cash disappears → losing interest. Must subtract R_f.
Long stocks onlyYESCash used to buy stocks → opportunity cost of interest.

SHARPE RATIO FORMULA:

Sharpe = (R_strategy − R_f) / σ_strategy

But if no funding cost: R_f = 0 in the formula

Gold arb: Sharpe = (R_GC_short + R_GLD − 2%) / σ

WORKED EXAMPLE: Why Futures = No Funding Cost

You have $1,000,000 in your brokerage account.

You buy 10 ES futures ($500K notional).

Cash used? $0 (futures = derivative contract)

Your $1M is still in the account as margin.

$1M × 2% interest = $20,000/year earned

Strategy return = futures P&L

Total return = futures P&L + $20,000 interest

→ Interest already included → don't subtract R_f

Contrast with stocks:

You have $1,000,000 in your brokerage account.

You buy $1M of SPY.

Cash used? ALL OF IT. $1M → SPY shares.

Interest earned on cash? $0.

Opportunity cost = $20,000/year.

→ Must subtract R_f from strategy return

💡 PRACTICAL NUANCE: In a long/short stock strategy, credit interest (from short sales) ≈ debit interest (from long purchases). Not exact — typically ~75 bps difference. But close enough to ignore for Sharpe purposes.

🎯 KEY TAKEAWAYS

1. Futures total return = spot return + roll return (γ). Spot is white noise (~2%/yr corn), roll is persistent (~24%/yr). The SIGN of γ holds for weeks/months/years because its drivers (storage, tariffs, macro) change slowly. This persistence = momentum.

2. Estimate γ from the forward curve: plot log(price) vs time-to-expiry → slope = −γ. Always scatter-plot first (cardinal sin: skip this). Backwardation = positive γ (fwd curve slopes DOWN). Contango = negative γ (fwd curve slopes UP). Sign flips!

3. To isolate pure roll return, hedge spot with an ETF that holds PHYSICAL assets (GLD for gold, XLE for oil). NOT futures-based ETFs (USO) — those cancel your roll return. Match timestamps: GC settles at 1:30 PM but GLD closes at 4:00 PM.

4. Subtract risk-free rate from Sharpe ONLY when the strategy has funding cost (buying stocks/ETFs). Futures-only strategies earn interest on margin → no funding cost → don't subtract. Use causal modeling (not data mining) to predict momentum.

THE MASTER FORMULA: Total Futures Return = Spot Return (α, white noise) + Roll Return (γ, persistent) — Estimate γ = −slope of log forward curve — When γ > 0 (backwardation): go long. When γ < 0 (contango): go short.

THE ONE THING TO NEVER FORGET: The sign of roll return persists because its drivers (storage costs, transportation, tariffs, geopolitics) change slowly. Time is your friend — roll return accumulates while spot noise cancels out. This is the causal model of futures momentum.
