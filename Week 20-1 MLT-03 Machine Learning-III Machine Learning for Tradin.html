Machine Learning III — Neural Networks & Text Mining

A neuron is just a = Σ wᵢxᵢ + b → φ(a). Stack them into layers and the network learns the weights by gradient descent + backpropagation. Built & back-tested on real EUR/USD, plus turning text into numbers with TF-IDF.

1 The artificial neuron

Copies a brain cell: dendrites (inputs) → nucleus (weighted sum) → axon (activation) → output. Each input xᵢ has its own weight wᵢ.

a = Σᵢ wᵢxᵢ + b    y = φ(a)

The simplest neuron is the Threshold Logic Unit: y = 1 if a ≥ θ else 0 — a linear classifier (a decision line w₁x₁+w₂x₂=θ), the same idea as an SVM. A linear activation instead (y=a) makes it a regression.

2 Architecture — a deep feed-forward MLP

Three kinds of layer; information flows one way (forward):

Input layerthe feature values (one node per feature)
Hidden layer(s)neurons that transform signals; "deep" = many
Output layerthe prediction (class or value)
Universal approximator: one hidden layer can approximate any continuous bounded function (given enough units). But NNs over-fit easily — minimise the generalisation error, not the training error.

3 Activation functions — and the vanishing gradient

Real values (notebook): sigmoid(0)=0.5, tanh(0)=0, ReLU(−3)=0.
Vanishing gradients. The sigmoid's slope peaks at only 0.25 (at a=0) and collapses to ~0 for large |a| — real: sigmoid′(6)=0.0025. Back-propagated through many layers the gradient vanishes and learning stops.
Fixes: use ReLU (no positive-side saturation). The opposite failure — exploding gradients — is handled by gradient clipping & weight regularisation.

6 Gradient descent & backpropagation

Training = pick weights that minimise a loss. Gradient descent steps the weights in the negative-gradient direction until the derivative ≈ 0.

loss = Σᵢ (yᵢ − ŷᵢ)²  ·  w ← w − η·∂loss/∂w

7 Text mining & sentiment (TF-IDF)

Turn words into numbers: drop stop-words ("the/of/that"), then vectorise with TF-IDF. A word's weight ∝ log(total docs / docs with the word) — rare, distinctive words score high.
