THE PROBLEM
Markets are mostly efficient — the EMH has huge empirical support. Beating it is hard.
YOUR GOAL
Find exploitable structure, test it honestly, judge it against a benchmark before risking capital.
THE EDGE
Math + clean vectorized back-tests + a quality checklist separate real edges from noise.
CORE THREAD → finance (EMH) rests on math (returns, regression, risk), and Python (numpy/pandas/matplotlib) implements it. Worked on real eod_prices.csv (NIFTY, 962 bars).

1 · EMH — the moving benchmark

  • Fama (60s): prices ≈ random walk; past changes don't forecast.
  • Jensen (78): efficient if no rule on an info set earns persistent risk-adjusted excess profit.
  • Timmermann–Granger (04): depends on info set S, search tech T, model class M → EMH is a moving target.
💡 If you fully believe the EMH, don't trade algorithmically — index funds instead.

2 · Random walk & best forecast

S(t+1) = S(t) + ε,   E[ε | F_t] = 0
# best least-squares forecast:
Ŝ(t+1) = S(t)     # just today's price!

No predictable alpha from past prices alone. Charting indicators are "worthless" under a pure random walk.

3 · Prices vs returns — the test

px.corr(px.shift(1))    # 0.99  prices trivially correlate
r = np.log(px/px.shift(1))
r.corr(r.shift(1))      # ≈ 0   returns: the real test

Always know: are you on prices or returns? Prices autocorrelate ~0.99; returns are where efficiency is judged.

4 · Autocorrelation & the noise band

band = 1.96/np.sqrt(N)          # 95% confidence
# r_t = rho*r_(t-1) + noise
#   rho=0   -> pure random walk (efficient)
#   rho>0   -> exploitable structure

ACF bars inside the band = noise (efficient). Persistent bars outside = exploitable. A synthetic AR(1) ρ=0.30 lights up at lag 1.

The two pictures: efficiency test (left) & the back-test verdict (right)

Left: NIFTY return autocorrelations scatter near zero (no persistent sign) while the AR(1) ρ=0.30 series breaks the band at lag 1. Right: an OLS sign rule loses money even in-sample with no costs — a clean EMH lesson.

6 · Risk metrics (annualize with √252)

sd_a = r.std()*np.sqrt(252)     # ann vol  19.2%
sharpe = r.mean()*252 / sd_a    # Sharpe   0.63
dd = equity/equity.cummax() - 1 # drawdown
max_dd = dd.min()               # -38.4%

7 · The curse of asymmetry — why drawdown ≠ volatility

Recovering a loss needs a larger % gain than the loss. Volatility (an average) hides this entirely.

Drawdown5%10%20%50%90%
Gain to break even+5.3%+11.1%+25%+100%+900%
⚠ In a live drawdown you feel it exponentially. Back-test mechanisms can't "avoid" a drawdown that already shows in the back-test — they only stop execution.

8 · Vectorized back-testing — OLS AR(7) strategy

Whole-array transforms, no for loop. Build lag features → fit OLS → forecast → take the sign → shift to avoid look-ahead.

for k in range(1, 8): df[f"l{k}"] = df["r"].shift(k)   # 7 lag features
X = np.column_stack([np.ones(n)] + [df[f"l{k}"] for k in 1..7])
beta = np.linalg.lstsq(X, df["r"])[0]    # alpha + beta VECTOR
df["pred"] = X @ beta
df["pos"]  = np.sign(df["pred"]).shift(1)   # ← shift = no look-ahead!
df["strat"] = df["pos"] * df["r"]           # position × return
⚠ Look-ahead / foresight bias — the position for day t may use only data up to t−1. Forgetting the shift fakes a great back-test.

Result: long 572 / short 381 · Buy & Hold +54.3% vs OLS strategy −47.3% (in-sample, no costs — yet it loses: EMH in action).

9 · Strategy quality checklist (pre-flight)

Statistical edge — returns/forecasts differ from noise (the ACF test).
Economic edge — profit after financing & transaction costs.
Persistence / robustness — works out-of-sample, not one tiny set.
Implementability — liquid tradable instruments (an index isn't tradable directly).
Benchmark & cost of capital — beats the bar, not just >0.
Risk profile — drawdowns, volatility, tail behaviour.

Retail-friendly playground: FX — liquid, tight spreads, symmetric long/short, clean REST APIs, mean-reverting around macro regimes (EUR/USD in the lecture).

30-second glossary

EMH — efficient market hypothesis; prices reflect all info.
random walk — next price = this price + zero-mean noise.
autocorrelation (ACF) — correlation of a series with its own lags.
confidence band — ±1.96/√N; inside = noise.
rho (ρ) — AR(1) coefficient; >0 = exploitable structure.
simple vs log return — cumprod vs cumsum.
Sharpe — annualized mean ÷ annualized volatility.
max drawdown — worst peak-to-trough fall.
curse of asymmetry — 50% loss needs +100% to recover.
vectorized back-test — array transforms, no bar loop.
look-ahead bias — using future data; fixed by .shift(1).
OLS — ordinary least squares; np.linalg.lstsq.
