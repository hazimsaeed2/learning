2️⃣ STATIONARITY & THE ADF TEST — The Mathematical Foundation

WHAT IS STATIONARITY?

WHAT: A price series is stationary if it keeps reverting back to its mean over time — like a sine wave oscillating around a center line.

WHY: If a price always returns to the mean, you can profit by buying below the mean and selling above it. This is the foundation of ALL mean-reversion strategies.

KEY RULES:

  • Random walk is NOT stationary (mathematically proven)
  • Only applies to long-term properties (need ≥120–200 days of data)
  • Stock returns are naturally stationary; stock prices are usually not
  • Visual inspection is subjective and error-prone — use ADF test instead
  • Most price series are NOT stationary for long periods → that's why pairs trading exists

Analogy (Daniel Kahneman): Sports Illustrated Jinx — athletes on the cover perform great, then revert to average performance. That's mean reversion in action.

WHY ADF TEST > BACKTESTING ALONE

ADF TestBacktesting
Checks EVERY data pointOnly checks points where trades trigger (e.g., 7 out of 100)
Fast: yes/no in <1 secondSlow: evaluate Sharpe, returns, drawdown
Objective (mathematical)Sensitive to parameters

⚠️ DANGER: If you skip ADF and only backtest, you might conclude "stationary" based on 7 data points out of 100. That's like hiring someone after reading 7% of their resume.

ADF TEST — THE MATH (Dickey-Fuller Equation)

ΔP(t) = λ × P(t−1) + ε(t)

ΔP(t) = change in price at time t = P(t) − P(t−1)
λ (lambda) = regression coefficient (slope) between ΔP(t) and P(t−1)
ε(t) = random noise (epsilon)

ScenarioMeaningResult
λ = 0 (Null H₀)ΔP(t) = noise only (random walk)NOT stationary ❌
λ < 0 (Alt H₁)Inverse relationship: price up → next move downSTATIONARY ✅

WORKED EXAMPLE — Finding λ by Hand

Sample prices: 6, 4, 8, ...

P(t)P(t−1)ΔP = P(t)−P(t−1)Point (x,y)
464−6 = −2(6, −2)
848−4 = +4(4, +4)

Plot (P(t−1), ΔP(t)) → draw best-fit line
Slope of line from (6,−2) to (4,+4) = (4−(−2))/(4−6) = 6/(−2) = −3
λ = −3 → negative → stationary ✅

Then: t-statistic = λ / standard_error
Standard error = std_dev / √n
Compare t-stat with critical value table.

CRITICAL VALUE TABLE

If t-stat > critical → NOT stationary

💡 SELECTING CONFIDENCE LEVEL: 99% → very few assets qualify (like only hiring IIT+MIT+15yr experience). 90% → more assets qualify. In practice, 90% is good enough for selecting stationary assets. Don't go below 90%.

3️⃣ MEAN REVERSION STRATEGY — Bollinger Band on a Single Stationary Asset

ADF TEST IN PYTHON

import pandas as pd                    # Data manipulation
from statsmodels.tsa.stattools import adfuller  # ADF test

data = pd.read_csv(                    # Read price data
    'AUD_CAD.csv',                     # AUD/CAD currency pair
    index_col=0,                       # First column = index
    parse_dates=True)                  # Convert to datetime
# data has 3404 rows, columns: Open, High, Low, Close

adf_results = adfuller(                # Run ADF test
    data['Close'],                     # Pass 1D price series
    maxlag=1)                          # Only 1 lag term
                                       # (more lags → overfitting)

t_stat = adf_results[0]               # First element = t-stat
# t_stat = -2.697

# Compare with 90% critical value = -2.57
# -2.697 < -2.57 → STATIONARY at 90% confidence ✅

💡 maxlag=1: We only use 1 lag term (P(t−1)) to prevent overfitting. More lags = more parameters = higher chance of overfitting. Default regression type is 'c' (constant only).

BOLLINGER BAND MEAN REVERSION CODE

# Step 1: Moving Average & Std Dev (5-day window)
df['MA'] = df['Close'].rolling(5).mean()   # 5-day MA
df['Std'] = df['Close'].rolling(5).std()   # 5-day std dev

# Step 2: Upper & Lower Bands (0.5σ width)
df['Upper'] = df['MA'] + 0.5 * df['Std']  # Upper band
df['Lower'] = df['MA'] - 0.5 * df['Std']  # Lower band

# Step 3: Entry/Exit Signals
df['Long_Entry'] = df['Close'] <= df['Lower']   # Buy signal
df['Long_Exit']  = df['Close'] >= df['MA']      # Sell signal
df['Short_Entry'] = df['Close'] >= df['Upper']  # Short signal
df['Short_Exit']  = df['Close'] <= df['MA']     # Cover signal

# Step 4: Position Tracking (1=long, 0=flat, -1=short)
df['Pos_Long'] = float('nan')                   # Initialize
df.loc[df['Long_Entry'], 'Pos_Long'] = 1        # Mark entries
df.loc[df['Long_Exit'], 'Pos_Long'] = 0         # Mark exits
df['Pos_Long'] = df['Pos_Long'].ffill()          # Forward fill

df['Pos_Short'] = float('nan')
df.loc[df['Short_Entry'], 'Pos_Short'] = -1
df.loc[df['Short_Exit'], 'Pos_Short'] = 0
df['Pos_Short'] = df['Pos_Short'].ffill()

df['Position'] = df['Pos_Long'].fillna(0) \
               + df['Pos_Short'].fillna(0)       # Combine

# Step 5: PnL
df['Pct'] = df['Close'].pct_change()             # Daily return
df['Strat_Ret'] = df['Position'] * df['Pct']     # Strategy ret
df['Cum_Ret'] = (1 + df['Strat_Ret']).cumprod()  # Cumulative
# Result: $1 → $1.80 over ~8 years

4️⃣ PAIRS TRADING & COINTEGRATION — Fabricating a Stationary Spread From Two Trending Assets

THE CORE IDEA

WHAT: EWA and EWC are individually trending (not stationary). But the SPREAD (EWC − 1.5×EWA) IS stationary → mean-reversion strategy works on the spread.

WHY: Australia and Canada are both energy-driven economies. Their ETFs can diverge temporarily, and the lecture treats the spread between them as something that often oscillates enough to test for mean reversion.

HEDGE RATIO — Why 1.5?

import statsmodels.api as sm             # Statistical models

      # CRITICAL: Use an initial training window
      # (here: first 90 days) to prevent LOOK-AHEAD BIAS
x_train = ewa_prices[:90]               # First 90 days of EWA
y_train = ewc_prices[:90]               # First 90 days of EWC

# OLS (Ordinary Least Squares) regression
# NOTE: NO constant added → line passes through origin
model = sm.OLS(y_train, x_train).fit()  # y = mx (no c)
hedge_ratio = model.params[0]           # Slope = 1.5
# hedge_ratio = 1.5

# Create spread for ALL data
spread = ewc_prices - hedge_ratio * ewa_prices
# spread = EWC - 1.5 × EWA

WHY CONSTANT = 0 (No Intercept)?

ReasonExplanation
LogicalIf gold miners (GDX) go to $0, gold itself (GLD) should also be $0. Two similar assets → both zero at zero.
MathematicalEstimating 1 parameter (slope only) instead of 2 (slope + intercept) → less overfitting, more generalizable

WHY ONLY 90 DAYS? → LOOK-AHEAD BIAS

⚠️ LOOK-AHEAD BIAS: Using ALL data to compute hedge ratio, then backtesting on the SAME data = knowing tomorrow's budget result and trading today. The hedge ratio "knows" the future. Use only the FIRST 90 days (training data), then test on the remaining data.

How to pick 90? Not magical. Could be 100 or 120. Rule: not too short (unstable regression) and not too long (misses current relationship — like judging a friendship by what happened 10 years ago).

COINTEGRATION TEST (Engle-Granger Two-Step Method)

# Step 1: Calculate hedge ratio (already done above)
# hedge_ratio = 1.5

# Step 2: Create spread and run ADF test
spread = ewc_prices - hedge_ratio * ewa_prices

adf_result = adfuller(spread, maxlag=1)
t_stat = adf_result[0]    # = -3.09

# Critical value at ~90% = -2.67
# -3.09 < -2.67 → spread IS stationary
# → EWA and EWC are COINTEGRATED ✅
# → Deploy mean reversion on the spread!

TRADING THE SPREAD — Position Sizing

Spread = Y − HR × X
Long spread → Buy 1 share Y, Sell HR shares X
Short spread → Sell 1 share Y, Buy HR shares X

PAIR SELECTION — Where to Find Good Pairs

✅ BEST CANDIDATES

  • ETFs: EWA/EWC, GLD/GDX
  • Currencies: AUD/CAD, EUR/CHF
  • Same-sector stocks: Coca-Cola/Pepsi, American Airlines/Delta, BofA/JPM, HDFC/Kotak (India)

Why? Similar businesses exposed to similar economic factors give you a better chance of a spread that behaves in a mean-reverting way.

❌ POOR CANDIDATES

  • Commodities: Each has its own dynamics, hard to find pairs
  • USD/INR: RBI intervention distorts, INR depreciates 5–7%/yr systematically
  • Large cap vs small cap: Vastly different volatility → spread dominated by one side

KEY INSIGHT

Stocks offer many choices but lose cointegration easily → use a portfolio of multiple pairs. ETFs and currencies are more stable but have fewer choices.

Advanced: Index arbitrage — Bank Nifty vs its top 5–6 components (HDFC, Kotak, SBI, Axis). Trade deviations between index and its constituents.

HALF-LIFE — How to Choose Look-Back Period

WHAT: Half-life = time for the spread to complete HALF of one mean-reversion cycle (from peak deviation back to mean).

HOW TO USE: Set the Bollinger Band look-back window to a multiple of the half-life (e.g., 1×, 2×, 3× half-life).

import numpy as np                     # Numerical computing

# Half-life calculation from spread
spread_lag = spread.shift(1)           # Spread at t-1
delta_spread = spread - spread_lag     # Change in spread
# Regress delta_spread on spread_lag
# slope = coefficient from regression
# half_life = -log(2) / slope
half_life = -np.log(2) / slope         # In days
# Use half_life to set look-back period
# e.g., look_back = int(half_life) or 2 * int(half_life)

🎯 KEY TAKEAWAYS

1. ADF before backtest: Always run the ADF test BEFORE backtesting. It checks every data point for stationarity, while a backtest only evaluates the few points where trades trigger (e.g., 7 out of 100). A profitable backtest on a non-stationary series is a trap.

2. Cointegration ≠ Correlation: Two assets can be correlated (both going up) but NOT cointegrated (spread is trending). Always check cointegration (ADF on the spread), never rely on correlation alone — the instructor lost money learning this the hard way.

3. Look-ahead bias kills pairs strategies: Use an initial training window such as the first 90 days to compute the hedge ratio. Using ALL data for the hedge ratio and then backtesting on the same data is like trading tomorrow's news today — it creates unrealistically flattering results that disappear in live trading.

4. LTCM proves risk management > strategy: Nobel Prize winners with a brilliant convergence strategy lost $4.6B in 4 months because of over-leverage and insufficient capital buffers. The strategy worked; the risk management failed. Keep reserves so the worst months do not force you out of the market.

THE MASTER FORMULA: Stationary Series = ADF t-stat < Critical Value → Deploy Mean Reversion (buy below lower band, sell above mean) | Pairs: Spread = Y − hedge_ratio × X → ADF on spread → trade the spread, not individual assets.

THE ONE THING TO NEVER FORGET: Statistical arbitrage is NOT risk-free arbitrage. The word "statistical" means you're betting on PROBABILITY, not certainty. Spreads can diverge indefinitely during extreme events. Keep capital buffers, use stop losses, and diversify across multiple pairs — because even Nobel Prize winners go bankrupt without risk management.
