① CORE EXAMPLE — the clean one-shot read

import warnings, pandas as pd
warnings.filterwarnings("ignore")

df = pd.read_csv(
    "TCS.NS.csv",
    index_col=0,        # Date = the index
    parse_dates=True,   # read it as a date
    dayfirst=True)      # dd/mm/yyyy

df.loc[:, ["Open","Close"]].head()

One call does everything: no separate Date column, the index is a real DatetimeIndex, and the frame is already sorted in time.

② Display options

Tiny market numbers print in scientific notation. Set a float format; reset to keep the kernel fast.

CommandEffect
set_option("display.float_format", lambda x:f"{x:.8f}")1e-05 → 0.00001000
reset_option("display.float_format")back to scientific
set_option("display.max_rows",100)show all rows
reset_option("display.max_rows")back to 60 (default)
⚠ Tip: showing 100k rows eats namespace and slows the kernel — always reset_option after.

③ Warnings ≠ errors

import warnings
warnings.filterwarnings("ignore")

A warning (deprecation, future change) does not stop your code — it just clutters output. Filtering is a setting on the current kernel, not a fix to the code. An error stops execution; that you must fix.

Series vs DataFrame: df["Close"] → Series · df[["Close"]] → DataFrame. Every DataFrame column is a Series.

④ The trailing-space column trap (real exchange CSVs)

Symptom
Columns are 'Close ' (hidden space) → df["Close"] raises KeyError: 'Close'.
Diagnose
list(df.columns) shows ['Date ','Open ',…] — the spaces are visible.
Fix
df.columns = df.columns.str.strip() → lookups work again.

Slice rule (recap): lst[0:2] includes index 0 and 1, excludes 2. Count = stop − start. [2:] = from 2 to end; [:] = everything.

⑦ Candlesticks in Python

Each candle encodes 4 numbers: open, high, low, close. Body = open→close (green up / red down); the thin wick spans low→high. The lecture uses Plotly go.Candlestick; the same picture in matplotlib is shown here.

import plotly.graph_objects as go
go.Figure(go.Candlestick(
    x=df.index, open=df.Open, high=df.High,
    low=df.Low, close=df.Close)).show()
